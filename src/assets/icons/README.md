# LeadTapp Icons

Place PNG icons here with the following sizes:
- icon16.png (16x16)
- icon32.png (32x32)
- icon48.png (48x48)
- icon128.png (128x128)

## Quick Generation

You can use any image editor or online tool to create these.
Recommended: Use a target/bullseye icon in KW red (#B40101).
